# 🤝 Telegram Dating Bot

Бот для анонимных знакомств с Telegram Mini App интерфейсом.

## ✨ Особенности

- Анонимное общение без фото и личных данных
- Telegram Mini App для удобного интерфейса
- Случайный подбор собеседников
- Возможность менять собеседников

## 🚀 Установка

1. Клонируйте репозиторий
2. Установите зависимости: `pip install -r requirements.txt`
3. Замените `TOKEN` в `bot.py` на ваш токен
4. Замените `WEBAPP_URL` на URL вашего хостинга
5. Запустите: `python bot.py`

## 📱 Настройка Mini App

1. Разместите папку `webapp` на GitHub Pages
2. В @BotFather настройте Menu Button URL
